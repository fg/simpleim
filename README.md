# simpleim
Simple Instant Messanger over TCP with nodejs

Run server
#node main.js

Connect to client

nc|netcat :server_ip :server_port

Usage:
Login with username
#me:{yourusername}

To send message to user, write this command
#to:{nameofuser}

Then write it. 
If you want to switch user, you have to write to:{nameofuser} again
